# Introduction
This is an exploratory project integrating in a single platform a chatbot 
assistant, an online marketplace interface and analytics regarding solid 
waste in urban environments.


# Setup
Created with python 3.10.2.

## Create a virtual environment
At your working directory run `python -m venv .venv`.
Activate with `source .venv/bin/activate`.

## Use the requirements.txt with pip
`python -m pip install -r requirements.txt`.


# Development Instance on Render
To access the development instance on [Render](https://render.com/) simply 
follow the [link](waste-manager-qhpm.onrender.com) provided in the project's 
description (upper right part of the repository -might take up to a minute 
to initialize since the current tier utilized is the free one).
